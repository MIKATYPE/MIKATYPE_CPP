美佳のタイプトレーナ Ｖｅｒ２．０６．０１ Windows C++版ソースコード

　教育用のタイプ練習ソフト「美佳のタイプトレーナV2.06」のWindows C++版ソースコードです。
　このソースコードの著作権は今村二朗が所有します。
　改変、転載、他ソフトの使用など自由にお使いください。
　
注意事項

１．

Visual Studio 2022
Visual C++ 2022
64ビット版で動作確認済です

２．

Visual C++がインストールされていないパソコンで動作させるにはVisual C++ のランタイムライブラリをインストールする必要があります。

３．
Visual Studioでコンパイルするときにはプロジェクト名をMIKATYPEにして下さい。

４．
このコースコードでは成績ファイル書き込み時にエラーがあってもダイアログを表示しません。

５．
練習テキストを変更するときには練習テキスト文字列の最後にストッパー""を追加してください。
ただしJAVA版のソースコードではストッパーの""は必要ありません。

例

	string MIKA_oubun[]={
		"ability",
		"begin",
		"country",
		""
		};
